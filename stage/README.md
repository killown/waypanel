# Stage

This directory contains **experimental and unstable components** of Waypanel.

Anything inside `stage/` is **not production-ready**, may be incomplete, broken,
or removed at any time, and **is not officially supported**.

## Purpose

`stage/` exists as a **staging area** for features and integrations that are:

- under active development
- being tested or prototyped
- blocked by external limitations
- not yet suitable for general users

Nothing here should be considered stable.
